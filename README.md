# Проєкт YesCite — Система автоматизації бібліографічних посилань

## Крок 1. Опис предметної області
**YesCite** — це вебплатформа, призначена для автоматизованого створення, збереження та управління бібліографічними посиланнями (цитуваннями). Система дозволяє студентам та дослідникам генерувати списки літератури за стандартами (APA, MLA, ДСТУ тощо), зберігати їх у власні проєкти, а адміністраторам — підтримувати базу стандартів оформлення.
